Первая задача при изучении дп - расчет n-ого числа Фибоначчи

Подходы к решению:
1. Наивный - рекурсия
- Значения рассчитываются экспоненциальное кол-во раз - неэффективно
2. Рекурсия, но с мемоизацией (запоминанием уже расчитанных значений) - Top-Down
- Однако стэк может переполнится - StackOverflowError
3. Итеративный, последовательный расчет значений до данного с запоминанием расчитанных значений в массив - Down-Top
- Используется лишняя память
4. Итеративный, но с запоминаннием только двух последних значений для расчета следующего - Down-Top
- Самый эффективный способ
