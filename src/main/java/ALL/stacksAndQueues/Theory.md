Стэк - структурая, работающая по принципу LIFO (Last In - First Out)
последний пришел, первым вышел

Основные операции:
- push(element) — положить элемент на вершину стека
- pop() — удалить и вернуть элемент с вершины 
- peek() — посмотреть элемент на вершине, не удаляя 
- isEmpty() \ (isFull - если фиксированного размера) — проверить, пуст ли стек 
- size() — количество элементов

Есть две реализации: 
1. Массивом
2. Связным списком
