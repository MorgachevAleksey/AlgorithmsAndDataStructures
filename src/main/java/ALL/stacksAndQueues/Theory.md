Стэк - структура данных, работающая по принципу LIFO (Last In - First Out)
последний пришел, первым вышел

Основные операции:
- push(element) — положить элемент на вершину стека
- pop() — удалить и вернуть элемент с вершины 
- peek() — посмотреть элемент на вершине, не удаляя 
- isEmpty() \ (isFull - если фиксированного размера) — проверить, пуст ли стек 
- size() — количество элементов

Есть две реализации: 
1. Массивом - Реализуется на ArrayDecque
2. Связным списком - Реализуется на LinkedList

Реализация связным списком хранит элементы в структуре узла, который содержит:
- Сам элемент
- Ссылку на следующий узел

Очередь - структура данных, работающая по принципу FIFO (First In - First Out)
первым пришел, первым вышел

Основные операции:
- enqueue (offer / add) — добавить элемент в конец очереди 
- dequeue (poll / remove) — удалить и вернуть элемент из начала очереди 
- peek (element) — посмотреть элемент в начале, не удаляя 
- isEmpty() — проверить, пуста ли очередь 
- size() — количество элементов
